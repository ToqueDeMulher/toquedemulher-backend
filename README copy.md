# Backend do Projeto

Esta pasta contém **todas as funcionalidades do backend**: lógica do servidor, APIs, banco de dados e tudo relacionado à parte do servidor.

- Código fonte
- Testes
- Rotas e controllers
- Modelos de dados
- Configurações e scripts do backend

> Todas as alterações devem seguir as regras do repositório:
> - Nunca commit direto no `main`
> - Crie branches a partir de `develop`
> - Use pull requests para mesclar alterações
> - Utilize os templates de **issue** e **commit**

## ⚠️ Lembre-se ⚠️

Use um arquivo **`.gitignore`** para **não enviar informações sensíveis** para o GitHub.  

> Exemplo: variáveis de ambiente no `.env`, senhas, chaves de API, arquivos de configuração privados, etc.
